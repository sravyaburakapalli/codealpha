# Hangman Game

## Description
A simple text-based Hangman game built in Python. The player guesses one letter at a time to reveal a randomly selected word.

## Features
- Random word selection
- 5 predefined words
- Maximum 6 incorrect guesses
- Input validation
- Displays guessed letters
- Win/Lose messages

## Technologies Used
- Python 3

## How to Run
1. Clone the repository.
2. Open the project folder in a terminal.
3. Run:
   ```bash
   python hangman.py
   ```

If `python` is not recognized, use:
```bash
py hangman.py
```

- `random` module
